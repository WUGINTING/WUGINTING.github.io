平台概述

🎯 核心定位

本平台是一個本地化、高度可配置的智慧型 RAG（檢索增強生成）知識庫系統，專為企業用戶設計，支援：

  • ✅ 智能文件處理：多格式文件上傳與解析
  • ✅ 知識庫管理：規範化的知識存儲與組織
  • ✅ 向量化搜索：基於語義相似度的精準檢索
  • ✅ 智能問答：多模型支持的對話系統
  • ✅ 工作流自動化：可視化工作流配置與執行
  • ✅ 實體提取：結構化信息自動提取

🌟 主要優勢

優勢 說明
本地部署 完全本地化運行，數據不上雲，隱私安全
高度可配置 切割規則、提示詞、模型參數均可自定義
多模型支持 支持 OpenAI、Gemma、Qwen 等多種 LLM
實時處理 異步任務隊列，支持大規模文件批處理
可視化管理 Vue.js + Quasar 精美前端界面
REST API 完整的 API 接口，易於集成
