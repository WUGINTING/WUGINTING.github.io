图片目录说明

这个目录用于存放博客的图片文件。

使用方法

方式 1：使用统一图片目录（推荐用于通用图片）

将图片放在这个 images 目录下，然后在文章中使用：

示例：

方式 2：使用文章专属文件夹（推荐用于文章特定图片）

当 post_asset_folder: true 时，每篇文章会自动有一个同名文件夹。

步骤：

  1. 创建与文章同名的文件夹（如果不存在）
    • 例如：文章是 我的文章.md，文件夹就是 我的文章/
  2. 将图片放入该文件夹
  3. 在文章中使用相对路径：

示例：

  • 文章：source/_posts/LLM-as-a-Judge.md
  • 图片：source/_posts/LLM-as-a-Judge/diagram.png
  • 使用：![架构图](diagram.png)

支持的图片格式

  • JPG / JPEG
  • PNG
  • GIF
  • WebP
  • SVG

注意事项

  1. 文件命名：建议使用英文和数字，避免中文文件名
  2. 图片大小：建议压缩图片，加快加载速度
  3. Alt 文本：记得填写图片描述，有助于 SEO 和可访问性

图片优化建议

  • 使用工具压缩图片（如 TinyPNG、ImageOptim）
  • 大图片建议使用 WebP 格式
  • 保持图片宽度在 1200px 以内（适合博客显示）
